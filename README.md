## Description
[Nest](https://github.com/nestjs/nest) project integrated with [Suites](https://suites.dev/) created with the starter repository.

## Project setup

```bash
$ npm install --legacy-peer-deps
```

## Compile and run the project

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Run tests

```bash
# unit tests
npm run test
```
